# Exercícios – Operadores de Comparação em JavaScript

Este conjunto de exercícios foi desenvolvido em HTML e JavaScript com o objetivo de demonstrar o funcionamento dos principais operadores de comparação.

Cada arquivo possui uma interface simples onde o usuário insere dois valores e visualiza o resultado da comparação, além de informações sobre tipo de dado e explicações sobre o operador utilizado.

## Exercícios

Operador de Comparação (==)
--

- Arquivo: (==).html

- Este exercício demonstra o uso do operador de comparação ==.

- O usuário informa dois valores.

- O sistema realiza a comparação utilizando ==.

- Exibe o valor digitado, o tipo de cada variável e o resultado da operação.

- Mostra que o operador == compara apenas os valores, permitindo coerção de tipo.

- Objetivo: Compreender a diferença entre comparar apenas valores, mesmo que os tipos sejam diferentes.

 Operador de Comparação Estrita (===)
 -

- Arquivo: (===).html

- Este exercício apresenta o operador ===.

- O usuário informa dois valores.

- Um dos valores é convertido para número.

- O sistema realiza a comparação utilizando ===.

- Exibe os valores, seus tipos e o resultado final.

- Mostra que o operador === compara valor e tipo ao mesmo tempo.

- Objetivo: Entender a comparação estrita e a importância do tipo de dado em JavaScript.

 Operador Estritamente Diferente (!==)
 -

- Arquivo: (!==).html

- Este exercício demonstra o operador !==.

- O usuário informa dois valores.

- O sistema realiza a comparação utilizando !==.

- Exibe os valores, os tipos e o resultado.

- Mostra que o operador verifica se os valores são diferentes considerando também o tipo.

- Objetivo: Compreender como funciona a comparação de desigualdade estrita.

 Operador Maior que (>)
 --

- Arquivo: (maior que).html

- Este exercício apresenta o operador >.

- O usuário informa dois valores numéricos.

- Os valores são convertidos para número.

- O sistema verifica se o primeiro valor é maior que o segundo.

- Exibe a operação realizada e o resultado.

- Objetivo: Entender comparações numéricas utilizando o operador maior que.

 Operador Maior que ou Igual (>=)
 --

- Arquivo: (maior ou igual a que).html

- Este exercício demonstra o operador >=.

- O usuário informa dois valores numéricos.

- Os valores são convertidos para número.

- O sistema verifica se o primeiro valor é maior ou igual ao segundo.

- Exibe a operação e o resultado da comparação.

- Objetivo: Aprender a utilizar comparações que envolvem igualdade e maioridade ao mesmo tempo.

Objetivo Geral
--

- O conjunto de exercícios tem como finalidade:

- Praticar operadores de comparação em JavaScript

- Entender a diferença entre comparação simples e comparação estrita

- Observar como o tipo de dado influencia o resultado

- Visualizar na prática o comportamento dos operadores relacionais
